# What is KnightZip?
**KnightZip** is a UX based zipping tool which provides you with the most strongest encryption method possible. It will offer to create keys for you within the range of **16** - **82** characters long, making sure nothing is getting inside the zip file besides the keyholder.

# How to use KnightZip?
It's pretty straight forward. First, open up **KnightZip** and drag a file or folder to the UX. You'll notice that it pastes the path of the file (or folder) you dragged.

After that, just press enter and then choose how long you want your key to be, press enter again and it'll begin zipping.
