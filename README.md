## What is KnightZip?

**KnightZip** is a GUI zipping tool built around strong encryption. It generates keys between **32** and **128** characters long, keeping your zipped files locked down to whoever holds the key.

No bloat. No nonsense. Just encrypt and go.

---

## Usage

> Drag and drop a file or folder onto the window, or use the **Browse** button.

| Step | Action |
|------|--------|
| 1 | Open **KnightZip** |
| 2 | Drop a file or folder onto the window, or click **Browse** |
| 3 | Adjust your desired key length using the slider |
| 4 | Hit **Encrypt & Zip** |
| 5 | Copy your generated key from the tool and store it somewhere safe |

> [!WARNING]
> Your key is only shown once inside the tool. Copy it before closing.
